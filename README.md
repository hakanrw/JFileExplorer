# JFileExplorer
File Explorer written with Java and Swing.

![image](https://github.com/hakanrw/JFileExplorer/assets/57678928/671409a5-fe55-4679-8f45-50d24e763bbc)



## Running the program
Run the project with `./gradlew run` (UNIX), or `gradlew.bat run` (NT)
